## BI 图表设计

### 图表核心模块

- 配置：负责图表的数据配置和样式配置
- 渲染：负责图表的渲染与更新
- 数据请求：负责图表的数据请求、请求参数处理以及数据响应处理
    - 内部数据请求
    - 外部数据请求
- 交互：负责图表的交互模块，如联动、跳转、预警、定时刷新以及下载数据等
    - UI 交互
    - 接口交互
- 权限控制

### 配置模块

- 样式配置
    - UI配置
    - json schema
- 数据配置

### 渲染模块

- 渲染引擎
    - echarts图表引擎
    - g2 图表引擎
    - 自定义渲染引擎
- 渲染配置
- 更新流程

### 数据请求模块

- 内部数据请求
- 外部数据请求
- 请求参数处理（钩子函数）
- 接口数据处理（钩子函数）

### 交互模块

- 插件集成
    - UI模块集成
    - 数据配置模块集成
